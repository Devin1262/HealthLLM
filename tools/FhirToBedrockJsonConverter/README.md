# Quick Usage Instructions

1. Build console app (.NET 8 SDK needed)
2. In terminal, navigate to exe
    ex. `cd  C:\Repos\CS6440_Group8\tools\FhirToBedrockJsonConverter\bin\Debug\net8.0>`
3. point arg 1 to whatever directory you extracted the Synthea data
4. point arg 2 wherever you please
    ex. `./FhirToBedrockJsonConverter.exe "C:\fhir\testdata" "C:\fhir\result\data.jsonl"`
